# Hardware

Technical specifications and capabilities of Board hardware.

## Display

- Screen Size: 23.8" diagonal
- Resolution: Full HD 1920×1080p
- Refresh Rate: 60Hz
- Panel Type: Wide viewing angle LCD
- Touch Sensor: Optically-bonded, high-resolution capacitive touch
- Coating: Anti-glare

## Touch Input

Board features an optically-bonded, high-resolution touch sensor integrated with the display. The anti-glare coating ensures visibility in various lighting conditions while maintaining touch responsiveness.

For implementation details, see Touch Input.

## Processing

| Component | Specification |
| --- | --- |
| Processor | MediaTek Genio 700 Octa-Core |
| CPU Cores | 8 cores (2× Cortex-A78, 6× Cortex-A55) |
| GPU | Mali-G57 |
| RAM | 4 GB |

## Storage

- Internal Storage: 64 GB embedded storage
- External Storage: SD card slot for additional storage

## Connectivity

### Wireless

| Feature | Specification |
| --- | --- |
| WiFi | MediaTek MT7921 WiFi 6 (802.11ax) dual-band MIMO |
| Bluetooth | 5.2, compatible with peripherals |

### Ports

- USB-C Accessory Port - For external camera and sideloading
- HDMI Output - For external display/sideloading
- SD Card Slot - For additional storage

## Audio

- Speakers: 2× 3W built-in speakers
- Microphone: 2× built-in microphones

## Physical Dimensions

| Attribute | Value |
| --- | --- |
| Dimensions | 25.7" × 13.9" × 1.5" |
| Faceplate | Customizable, removable wood-finished frame |

## Power & Environmental

- Power Supply: Wall-powered via power adapter (19V / 45W)
- Ingress Protection: IP4X rating (protected against solid objects >1mm)
