# BoardPauseButtonIcon Enum

Namespace: Board.Core

Specifies the icon type for a button in the Board pause screen.

```csharp
public enum BoardPauseButtonIcon
```

| Field | Value | Description |
| --- | --- | --- |
| None | 0 | No icon. |
| CircularArrow | 1 | A circular arrow icon (e.g., restart). |
| Square | 2 | A square icon (e.g., stop). |
| LeftArrow | 3 | A left-pointing arrow icon (e.g., back). |
| DoorWithArrow | 4 | A door with arrow icon (e.g., exit). |
