# Board Unity SDK — Documentation Mirror

Local mirror of the Board **Unity SDK** documentation from <https://docs.dev.board.fun/>,
captured for offline reference while porting the Trafalgar prototype.

- SDK version documented: **Unity v3.3.0** (April 3, 2026 changelog entry)
- Captured: 2026-06-04
- Total pages: **63** saved, **0** failed (plus this INDEX = 64 files).

Every file begins with a `> Source:` line linking the original URL. Cross-SDK guide
pages (`guide-*.md`) keep the Unity samples as the primary target and retain the
Godot/Web snippets for reference. The API reference is DocFX-generated; `api-*.md`
files are namespace landing pages and `api-class-*.md` files are per-type pages.
