## Getting Help

### Discord
Join the Board Developer Discord for direct support and community discussion.

### Email
For general inquiries, contact us at hello@board.fun.

When reporting an issue, include:
- OS version (Settings > System > Firmware)
- Which SDK and its version (Unity, Godot, or Web)
- Engine or runtime version (Unity Editor or Godot version)
- Steps to reproduce
- Expected vs actual behavior
- Relevant logs or screenshots
