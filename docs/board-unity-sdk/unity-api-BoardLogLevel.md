# BoardLogLevel Enum (Board.Core)

Specifies the level of logging in the Board SDK.

```csharp
public enum BoardLogLevel : System.UInt32
```

### Fields

- `None` = 0 — Disables all logging.
- `Debug` = 1 — Enables debug-level logging.
- `Info` = 2 — Enables info-level logging.
- `Warning` = 4 — Enables warning-level logging.
- `Error` = 8 — Enables error-level logging.
- `All` = 15 — Enables all logging levels.
